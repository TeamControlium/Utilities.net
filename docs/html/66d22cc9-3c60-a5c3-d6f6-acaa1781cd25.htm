DetokeniserCustomTokenProcessor Property

[This is preliminary documentation and is subject to change.]

Gets or sets delegate for processing custom tokens if required. If set, delegate is called before internal token processing to allow overriding if required.

public static Func<char, string[], string> CustomTokenProcessor { get; set; }

Property Value

Type: FuncChar, String, String
Remarks
Assigned delegate must take char (delimiter being used) and the token (split into an array using delimiter as separator) and return the resolved token text. If the token cannot be resolved, delegate should return a null. Any exception should be allowed to ripple up to enable the Utilities Detokeniser to handle the error.
