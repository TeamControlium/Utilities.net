Gets reference to the Dynamic items class allowing indexed access to and setting of all Globally stored data items.

Remarks
This is a reference to an indexer class. So can be used as an indexed property. All data stored in the Global repository are visible across all threads.
Examples
Example stores string 'some data' in the repository, naming it 'MyItem'. Then recalls Repository item 'MyItem', storing it in myData.
Copy
Repository.ItemGlobal["MyItem"] = "some data";
string myData = Repository.ItemGlobal["MyItem"]; // myData now equal to 'some data'
